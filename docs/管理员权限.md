# 管理员权限说明（上送设置 / 字段映射）

「上送设置」和「字段映射」两个 tab 默认隐藏,只有**管理员**能看到和修改。
权限基于 **Ed25519 离线签名令牌**,不依赖任何本地白名单文件 —— 使用者改不了自己的权限。

---

## 原理（为什么使用者改不了）

- 权限的"真相"在**分发者的私钥**里,不在使用者磁盘上。
- 软件只内置**公钥**(`src/admin-pubkey.js`),公开无妨:公钥只能验签、不能签发。
- 没有私钥就**伪造不出**有效令牌;改本地任何文件都无效(验签不通过)。
- 令牌**绑定系统用户名**(大小写不敏感)→ 不能转借给别人用。
- 令牌带**过期时间** + 解锁后只在**内存**开会话 → 到期失效、程序重启也失效。

> 唯一防不住的极端情况:使用者直接改 JS 源码删掉权限判断。纯源码分发无法 100% 防住,
> 需要时可用 pkg 打包成二进制抬高门槛(见 README/PROJECT-NOTES,待实现)。

## 分发者:如何签发临时管理员令牌

> 前提:`secrets/admin-private-key.pem` 在你本机(首次已生成)。**此文件绝不能分发、绝不入库。**

### 方式一:面板可视化签发(推荐)

只有**本机存在私钥**时,面板右上角才会出现 **🔑 签发令牌** 按钮(分发副本无私钥 → 不显示、调接口也 403)。

1. 右上角点 **🔑 签发令牌**。
2. 填**使用者的系统用户名** + **有效小时**(默认 1)→ 点「生成令牌」。
3. 点「复制」把令牌发给对方。

> 这个签发能力天然只在你这台机器可用,不会随软件分发出去。

### 方式二:命令行签发

```bash
# 给某使用者签发 1 小时(默认)的管理员令牌
node scripts/gen-admin-token.js --user <对方的系统用户名>

# 指定有效期(小时,可小数)
node scripts/gen-admin-token.js --user zhangsan --hours 8
node scripts/gen-admin-token.js -u zhangsan -h 0.5
```

- `--user`:对方的**系统用户名**(大小写随意)。对方在面板右上角能看到自己的用户名。
- 把输出的**令牌字符串**发给对方即可。

## 使用者:如何解锁

1. 面板右上角点 **🔒 管理员**。
2. 粘贴分发者给的令牌字符串。
3. 解锁后出现「上送设置」「字段映射」两个 tab,右上角显示**剩余时间倒计时**。
4. 到期 / 关程序 / 点倒计时旁的 ✕ 即退出管理员。

## 密钥轮换（令牌泄露或定期更换时）

```bash
node scripts/gen-keypair.js
```

- 新私钥写入 `secrets/admin-private-key.pem`(旧的自动备份成 `.bak-时间戳`)。
- 把打印出的**新公钥**复制进 `src/admin-pubkey.js` 替换原公钥。
- 重新打包分发。**旧私钥签发的所有令牌立即失效。**

## 后端保护点（防绕过前端）

光隐藏前端 tab 不够,后端也做了校验:

- `PUT /api/settings`(改地址/Token/字段映射等)→ 需管理员会话,否则 403。
- `GET /api/settings` → 非管理员**不下发 apiToken 明文**(掩码)。
- 上送报文详情(`GET /api/queue/:qid`):
  - `apiToken` **始终掩码**(`••••••（仅管理员可见）`),任何人都看不到明文。
  - **上送请求地址**(`target`)→ 非管理员掩码,仅管理员可见真实地址。

会话靠请求头 `x-admin-session` 传递(前端解锁后自动带上),会话仅存服务端内存。

- `secrets/`(管理员私钥)
- `scripts/gen-admin-token.js`、`scripts/gen-keypair.js`(签发/造钥脚本)

## 想让某台 dist 副本也能签发令牌（显示 🔑 按钮）

🔑 签发按钮的出现条件是:**该机器的项目根目录存在 `secrets/admin-private-key.pem`**
(后端 `GET /api/admin/issuer` 检测此文件)。所以默认分发包没有私钥 → 不显示按钮、签发接口 403。

若你想在另一台你信任的机器上也能签发,只需把这两样拷进该副本的项目根:

```
secrets\admin-private-key.pem      ← 必需（签发就靠它）
scripts\gen-admin-token.js         ← 可选（想用命令行签发才需要）
scripts\gen-keypair.js             ← 可选（想在该机轮换密钥才需要）
```

> ⚠️ 拷过去 = 把签发能力也给了那台机器(等于多一把私钥副本)。只往**你自己掌控**的机器拷,
> 普通使用者的电脑**绝不要**放 `secrets/`。拷贝后重启服务、刷新页面,🔑 按钮即出现。
